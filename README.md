# TCC — Laboratórios Didáticos de Segurança Web

Repositório com ambientes simulados para estudo e demonstração de vulnerabilidades web e suas contramedidas. Todos os laboratórios rodam localmente no navegador, sem necessidade de backend ou banco de dados.

## Simulações disponíveis

| Pasta | Tema | Descrição |
|-------|------|-----------|
| [`XSS/`](XSS/) | **Cross-Site Scripting** | Injeção de scripts maliciosos (Refletido, Persistente e DOM) e a contramedida de Sanitização. |
| [`CSRF/`](CSRF/) | **Cross-Site Request Forgery** | Requisições forjadas a partir de um site malicioso e a contramedida com Token Anti-CSRF. |

Cada pasta possui seu próprio `README.md` com a explicação detalhada, os cenários de teste e o código da contramedida.
